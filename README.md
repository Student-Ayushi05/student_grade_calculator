# Student Grade Calculator
## Project Overview
This is a python program which takes students marks and return grades with an encouraging messages using grading logic and some coding skills.

## Project objectives
The main objective of this project is to:
 
  - use conditional statements(if-elif-else) to make decisions in program.
  - use while loop to handle invalid inputs.
  - calculates grades of no. of students at a time.
  - implement grading logic based on student marks.
  - create at least one function.

 ## Technical Requirements
- Python 3.14.5
- vs code

## How to Run:
- Download the repository
- Open terminal
- Navigate to the project folder
- Run:

  python grade_calculator.py

## Code Explanation
The program uses:

- input() to collect information from the user.
- use conditional statements(if-elif-else) to make decisions in program.
- use while loop to handle invalid inputs.
- uses grading logic which is provided in the project
- print() statements to display a personalized encouraging message.
- use Exception handling(try-except) to handle errors gracefully without crashing.
- use return to sends value back form a function.
- use function(def) to organize a grading logic.
